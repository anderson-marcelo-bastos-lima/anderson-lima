About The Project

The Gryffindors Portfolio project was an exercise in evaluating each step necessary to design a website project, design the wireframe document, prepare the directory structure with CLI, create the repository on GitHub, develop the page, and publish it to GitHub Pages.

It is an excellent experience to learn to follow the steps and to assimilate how each step is essential in elaborating a project from its conception to its implementation.

The project was a great challenge to put concepts into practice in a real case.
