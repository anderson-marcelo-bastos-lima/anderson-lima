About The Project

The Task Planner project was developed in a group of three participants, where it was necessary to understand the project scope requests, design the wireframe document and create the pages.

Group work only works if we manage to plan, divide activities properly among group members and create adequate communication channels.

The project was an excellent activity for us to use communication, planning and teamwork skills. This project's biggest challenge was reaching the goal as a group.
