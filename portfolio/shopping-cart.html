About The Project

The Shopping Cart project was a code challenge to develop an application with the concepts of e-commerce with a list of available products and automatic calculation of the total price of selected products.

In 2019, version 16.8 of React was released with the addition of Hooks. They allow you to use state and other React features without writing a class. The Shooping Cart application was built using React Hooks, Javascript, Bootstrap and HTML.

It was an adventure to develop an application in React with the concepts of e-commerce following a wireframe document. Hope you have a great experience navigating this project.
